# StreamUtility.CopyToMemoryStream method

Copies the contents of *stream* to a new MemoryStream.

```csharp
public static MemoryStream CopyToMemoryStream(Stream stream)
```

| parameter | description |
| --- | --- |
| stream | The stream to copy. |

## Return Value

A MemoryStream containing all the data read from the stream. The caller is responsible for disposing this stream.

## Remarks

The returned MemoryStream will be positioned at the first byte.

This method is useful when *stream* does not support seeking, but a seekable stream is required.
