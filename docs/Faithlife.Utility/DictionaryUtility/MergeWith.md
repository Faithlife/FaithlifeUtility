# DictionaryUtility.MergeWith&lt;TKey,TValue&gt; method

Merges *otherDictionary* into *thisDictionary*. The *mergeWithStrategy* specifies how to resolve any key conflicts, either taking the new value, retaining the original value, or throwing an exception.

```csharp
public static void MergeWith<TKey, TValue>(this IDictionary<TKey, TValue> thisDictionary, 
    IDictionary<TKey, TValue> otherDictionary, MergeWithStrategy mergeWithStrategy)
```

| parameter | description |
| --- | --- |
| TKey | The type of the key. |
| TValue | The type of the value. |
| thisDictionary | The dictionary to modify. |
| otherDictionary | The dictionary to merge from. |
| mergeWithStrategy | Specifies how to handle key collisions. |

## Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | If there is a key conflict and *mergeWithStrategy* is MergeWithStrategy.ThrowException. Note that some KeyValuePairs from *otherDictionary* may have already been added to *thisDictionary*. |
